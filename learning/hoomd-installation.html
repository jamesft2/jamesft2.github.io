Hoomd Installation

HOOMD-blue is the molecular dynamics software we primarily use in our group

Installing HOOMD If you have a linux or mac computer, you can install the software on your computer with these instructions. https://hoomd-blue.readthedocs.io/en/stable/installation.html

If you have a windows computer, you will need to either set up a virtual Ubuntu environment or dual boot your computer to do this. The process to install wsl (a linux subsystem which lives on your computer) is discussed here (and a few other places): https://cs107e.github.io/guides/install/wsl-setup/

Other packages

In your conda environment, also install (either now or when you need them): pandas numpy matplotlib seaborn scikit-learn scipy ipykernel gsd ovito (sometimes this is finicky) freud
