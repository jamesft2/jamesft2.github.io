Ovito Visualization

OVITO is an open source particle-based visualization software which you can download here: OVITO

In Gaussian Chain, there is a tutorial for using OVITO to view a snapshot. Here, I will document several functions I use frequently in ovito as a reference.

Nowadays, I use mostly the python interface for OVITO for a few reasons: it can be automated, it can be used to calculate quantities in workflows, and you can access all the premium ovito features as well. Nothing can replace opening and orbiting around an object

Opening a file

File -> Open -> select .gsd or .xyz file

Select particle types -> Select type of interest -> choose color of your choice

Using Modifiers

Modifiers are how you can do any type of analysis or change the visualization. They are applied sequentially, and can be grouped together and/or hidden.

Changing Cross Sections

To do this (say you want to show off a slightly different part of the box). First, select affine transformation. Uncheck simulation box. Below translation, type the amount you want to shift the cell. Then, select wrap at periodic boundary

Making particles transparent

Compute property modifier -> Output Property: Transparency -> Value 1 = Transparent, 0 = Opaque. Can 'select compute only for selected element.'

Making a Surface Mesh

There are two methods. Alpha-shape method (which takes a sphere of size probe-shere radius) and tries to find where the boundaries of where it can exist or didn't. Gaussian density method (which depends on particle size) treats each particle as a gaussian density and if the density at a given position is greater than the threshold it is inside the surface. I prefer alpha shape method. The probe sphere size should be at minimum 1.1 times the particle radius. The smoothing level changes t eh properties you are resol ving. I have used 1, 6, 10, 50, and 100 for various applications

Saving a .png or .pdf

Click the camera button in the top right corner. Either select an output location (with selecting save to file) or don't. Then select render active viewport. Either save the folder before hand or don't

Saving a .gif

Select Complete animation. Check save to file. Under choose -> select a filetype of either .mov or .gif. Select render active viewport.

Saving a surface mesh

When a surface mesh is in your pipeline, File -> Export -> select .vtk triangle mesh as the type.
